# 部署说明

## 方案 A：Vercel 快速部署

适合快速验证、海外访问、无需备案的阶段。

```bash
cd pair-aura-web
npm i -g vercel
vercel --prod
```

部署后会得到一个 `https://xxx.vercel.app` 链接。

优点：

- 很快；
- 不需要服务器；
- 可直接拖拽或 Git 部署；
- 适合测试和原型。

注意：

- 中国大陆访问速度和稳定性可能不如国内云；
- 若要面向中国大陆大规模用户，建议使用国内云厂商并完成备案。

## 方案 B：腾讯云 / 阿里云静态网站

适合正式面向中国大陆用户。

步骤：

1. 买域名；
2. 购买中国内地云服务或静态托管；
3. ICP 备案；
4. 上传 `pair-aura-web` 文件；
5. 配置 HTTPS；
6. 将域名放入海报二维码和分享链接。

## 方案 C：Cloudflare Pages / Netlify

适合海外访问和免费测试。

优点：免费、部署简单。

注意：中国大陆访问不一定稳定。

## 正式上线前必须补充

- 隐私政策；
- 使用条款；
- 免责声明；
- 第三方商标说明；
- 数据删除机制；
- 若接入 OCR/AI，需要图片上传授权和删除入口。
