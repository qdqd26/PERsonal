# Pair Aura V10 单文件版

V10 是基于 V9 的 UI / 使用流程重构版，继续保持单文件部署：只需要上传 `index.html` 到 GitHub Pages 即可运行。

## 这版重点

- 首页只保留「开始生成报告」和「看一个示例」。
- 官网测评入口降级到输入页的辅助入口，避免打断主流程。
- 首页文案改为：不是给你贴标签，是看你和 TA 会发生什么。
- UI 从重型海报感调整为更轻、更精致的年轻社交工具感。
- 保留 V9 的算法、SBTI/MBTI 输入、文本素材库、报告页和分享海报。

## 部署方式

1. 只上传 `index.html`。
2. Commit 到 GitHub。
3. 等待 GitHub Pages 自动部署。
4. 用无痕窗口打开页面测试。

## 上线后测试

- 首页是否显示深色 Neon Aura UI。
- 点击「看一个示例」是否直接生成报告。
- 点击「开始生成报告」是否跳到输入区。
- 输入类型后是否生成报告。
- 复制链接是否能还原报告。
- 下载海报是否正常。
