# Pair Aura 人格匹配度报告

Pair Aura 是一个单文件静态网页，用来把两个人已有的 MBTI / SBTI / 九型人格结果转化成一份双人关系说明书。

它不是心理诊断，也不是陌生人匹配平台。它的目标是让用户快速看到：

- 你们哪里同频；
- 哪里互补；
- 哪里容易互相触发；
- 应该怎么更舒服地相处或合作；
- 是否愿意把报告发给对方。

## 当前形态

- 单文件部署：只需要 `index.html`。
- 无后端、无登录、无数据库。
- 支持 MBTI、A/T、SBTI、九型人格。
- 生成合拍指数、关系类型、关键词、六个关系维度、四个主要差异、六张关系说明卡。
- 支持复制报告链接、复制分享文案、生成二维码海报。
- 支持匿名事件接口预留，但默认不会向任何接口发送数据。

## 使用方式

直接双击 `index.html` 即可预览。

更接近线上环境：

```bash
npx serve .
```

然后打开终端显示的本地地址。

## GitHub Pages 部署

1. 上传并替换仓库里的 `index.html`。
2. Commit changes。
3. 等待 GitHub Pages 自动刷新。
4. 用无痕窗口打开线上地址测试。
5. 如果仍显示旧页面，使用 `Ctrl + F5` 强制刷新。

## 上线检查

- 首页是否正常展示深紫渐变、光球和玻璃卡片。
- A/B 输入区是否能选择 MBTI、SBTI 和九型。
- 不填九型是否也能生成报告。
- 生成报告后是否显示合拍指数、关键词、维度条和六张说明卡。
- 复制链接后是否能在新窗口还原同一份报告。
- 下载海报后二维码是否能扫码打开报告。
- 手机端是否没有横向溢出。

## 隐私与边界

页面不会默认上传任何用户数据。匿名统计只有在代码中配置 `ANALYTICS_ENDPOINT` 且用户未取消授权时才会发送。匿名数据不包含昵称原文、手机号、微信号、头像、聊天记录、原始截图或精确地理位置。

## SBTI 数据修正

已将 SBTI 下拉选项修正为当前完整的 27 种 SBTI 类型，并把旧版自定义标签映射为最接近的正式类型，避免旧链接直接失效。
