# Pair Aura V13 单文件版

V13 基于 V12 继续优化两件事：

1. **语言素材库提质**：移除“说人话就是”等 AI 感强的表达，改成更温和、更有人情味、更适合发给朋友/情侣/搭子的短句。
2. **分享链路增强**：报告链接改成 URL-safe 编码；海报中绘制真实二维码，优先保障用户能复制链接到微信，或下载可扫码海报。

## 使用方式

GitHub Pages 直接替换仓库里的 `index.html` 即可。

## 保留特性

- 单文件部署，不依赖 `styles.css` / `app.js`。
- 支持 MBTI、A/T、SBTI。
- 关系组 2 个：友谊/情侣、搭子/学习/项目。
- 语气 2 个：温情吐槽、现实建议。
- 每份报告固定 6 张特质细节评价卡。
- 关系签文库：古代诗词 22 条 + 现代歌单签 24 条。
- 现代歌单签只使用歌名、歌手和关系意象解释，不引用现代歌词原文。

## 分享说明

- 点击“复制报告链接”可把当前报告链接复制到微信。
- 点击“复制分享文案”可复制一段完整分享文字。
- 点击“发给 TA”会优先调用系统分享，失败时自动复制分享文案。
- 下载海报会在海报右下角生成二维码；如果二维码库加载失败，仍可复制链接分享。
